# bulwark ^_____^

> minimal terminal project in python

bulwark is a small cli project focused on terminal workflow, persistent state, and multithreaded execution.

## overview

the project combines:
- a terminal interface
- multiple run modes
- local config and progress files
- logging
- worker threads and queues

it is structured more like a compact tool than a one-off script.
